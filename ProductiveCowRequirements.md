# ProductiveCow types and variables

> Variables

`id` - nazwa krowy -> np.: iron cow, obsidian cow

`bucket_id` -> bucket dla danego typu krowy

`texture` - Przechowuje lokalizacje tekstury

`worldSpawn` -> boolean czy jest mozliwe zeby sie respiła naturalnie albo tylko za pomoca preedowania

`breedItem` -> item do karmienia / breedowania

`spawnEgg` -> jajko do spawnowania

`Health` -> domyślnie 10, albo dowolny wpisany
